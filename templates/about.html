About this application

This project proposes an application to be developed in which the user will view a video clip (without an audio). The system will auto gener- ate questions based on these videos watched by the user. The user has to answer the questions generated by the system in the form of MCQs. The questions generated and the answers to the MCQ will be recorded by the system real-time for further analysis. The video can be played “n” number of times before submitting the answer. The user’s answer will be matched to the system’s model answer and if there is a match the user is permitted to login otherwise the user will be directed to give the right answers by playing the next CAPTCHA video clip. This CAPTCHA sys- tem is using video clips based on Visual Question Answering technique.
